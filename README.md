"Remittance scams" (fake Gcash/PayMaya payments)
"Training center scams" (pay ₱5k-20k for fake certifications)
"Job placement fee scams" (illegal under Philippine law)
"Work-from-home package scams" (buy ₱15k starter kit)

PH Beginner Essentials:

Red flags in "Homebased/Online Jobs PH" Facebook groups

How to invoice properly
